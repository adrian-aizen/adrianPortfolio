# 🌟 Project Overview
This project is a full-stack web application that provides a user-friendly interface for users to navigate and interact with. The application is built using React for the frontend and Express.js for the backend, with a MongoDB database for storing data. The project includes features such as a contact form, navigation bar, and home page with user information. The application is designed to be responsive and interactive, with a focus on providing a seamless user experience.

## 🚀 Features
* Contact form with validation and email sending functionality
* Navigation bar with responsive design and client-side routing
* Home page with user information, including profile picture, bio, and skills
* API service for making requests to the backend server
* Backend server with CORS configuration and route handling for contact form submissions
* Email sending functionality using Nodemailer

## 🛠️ Tech Stack
* Frontend: React, React Router DOM, Tailwind CSS, PostCSS
* Backend: Express.js, Node.js, MongoDB, Nodemailer
* Database: MongoDB
* Build Tools: Webpack, Babel
* Dependencies: body-parser, cors, dotenv, express, mongoose, nodemailer

## 💻 Usage
To use the application, follow these steps:
1. Open a web browser and navigate to `http://localhost:3000`
2. Interact with the application by clicking on the navigation bar and filling out the contact form
3. Submit the contact form to send an email to the backend server
